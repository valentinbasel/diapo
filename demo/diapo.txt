DIAPO
-----

Diapo es un software para la creación de archivos SVG que se pueden 
utilizar como una forma relativamente facil y rapida de hacer presentaciones
basadas en el plugin 'SOZI' para inkscape.

SVG
===

SVG
---

El archivo resultante es un archivo SVG (standar vector graphic)
el cual puede ser editado con inkscape y reproducido en cualquier 
navegador moderno (firefox, google chrome).
De esta forma, la creación de presentaciones es relativamente sencilla
aunque no permite muchas alternativas, es mas bien un sistema para poder
hacer presentaciones rapidamente.

video1
======

Videos
------
El sistema soporta insertar videos (locales o url), sin embargo la 
opción de autoplay esta activada y todos los videos que esten en
DIAPO se van a ejecutar ni bien se inicie la presentación.
se puede deshabilitar autoplay, desde firefox.
yendo la pagina de config y buscar la opción autoplay para pnerla en 
false
